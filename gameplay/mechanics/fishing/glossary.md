# Fishing glossary

| Term                  | Meaning                                                                                                             |
| --------------------- | ------------------------------------------------------------------------------------------------------------------- |
| **Cast**              | Timed channel started on a selected adjacent water tile while a fishrod is equipped.                                |
| **Fishrod**           | Hotbar tool kind (`fishrod`) with tiered cast speed. Held overlay visual exists but is currently disabled globally. |
| **Liquid water**      | Unfrozen water tile resolved by the plaza water engine; ice blocks casting.                                         |
| **Catch**             | One stack of `world-plaza-fish` granted when the cast channel completes.                                            |
| **Cast range**        | Chebyshev distance ≤ **2** tiles from the player to the water tile center.                                          |
| **Inventory add SFX** | FilmCow strap tighten when fish enters the bag; via `notifyingWorldPlazaInventoryItemAdded`.                        |
